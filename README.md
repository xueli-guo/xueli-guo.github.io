# Xueli Guo

Personal academic website for Xueli Guo, hosted at <https://xueli-guo.github.io/>.

This site is built with Jekyll and the Academic Pages theme.

## Editing

- Site settings: `_config.yml`
- Homepage: `_pages/about.md`
- Navigation: `_data/navigation.yml`
- CV: `_pages/cv.md`
- Publications: `_publications/`

## Deployment

GitHub Pages rebuilds the site automatically after changes are pushed to the repository.
